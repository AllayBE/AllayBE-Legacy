# AllayBE
AllayBE is a MCBE server software

If you encounter any memory leaks, make a PR to fix it or report it in our Discord (https://discord.gg/8QndUrGFx2)

# Building

### Windows
```
cmake -DCMAKE_BUILD_TYPE=Debug
msbuild AllayBE.sln
```

### linux
```
cmake -DCMAKE_BUILD_TYPE=Debug
make
```
