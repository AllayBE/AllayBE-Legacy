# AllayBE
AllayBE is a MCBE server software

If you encounter any memory leaks, make a PR to fix it or report it in our Discord (https://discord.com/invite/BmQXY3vvvp)

# Building

### Windows
```
cmake -DCMAKE_BUILD_TYPE=Debug
msbuild AllayBE.sln
```

### linux
```
cmake -DCMAKE_BUILD_TYPE=Debug
make
```
